# @webpinnacles/web

Next.js 14 public website with App Router, ISR, and JSON-LD schema helpers.

## Run

```bash
pnpm --filter @webpinnacles/web dev
```

## Routes

- `/`
- `/services/`
- `/services/[slug]/`
- `/services/case-studies/`
- `/about/`
- `/blog/`
- `/blog/[slug]/`
- `/contact/`
- `/privacy-policy/`
